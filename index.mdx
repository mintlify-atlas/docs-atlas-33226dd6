  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Univerto in three simple steps
  </p>
  
  <Steps>
    <Step title="Install the package">
      Add Univerto to your project using your preferred package manager:
      
      <CodeGroup>
      ```bash npm
      npm install univerto
      ```
      ```bash yarn
      yarn add univerto
      ```
      ```bash pnpm
      pnpm add univerto
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Import a converter">
      Univerto provides modular exports for optimal tree-shaking. Import only what you need:
      
      ```typescript
      import { TimeUnitConverter, TIME_UNIT } from 'univerto/time'
      ```
      
      <Info>Each unit category has its own export path (e.g., `univerto/length`, `univerto/mass`) to keep your bundle size minimal.</Info>
    </Step>
    
    <Step title="Convert units">
      Use the fluent API to perform conversions with a clear, readable syntax:
      
      ```typescript
      const result = TimeUnitConverter
        .from(24, TIME_UNIT.HOUR)
        .to(TIME_UNIT.DAY)
        .convert()
      
      console.log(result) // 1
      ```
      
      <Tip>The conversion engine uses fraction-based calculations internally, avoiding floating point precision issues common in JavaScript.</Tip>
    </Step>
  </Steps>
</div>
